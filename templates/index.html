About the Project

Understanding the challenge of misinformation in Nepali media

The Problem

In today's digital age, fake news spreads rapidly through social media and online platforms, particularly in the context of Nepali political discourse. Misinformation undermines democratic processes, influences public opinion, and erodes trust in legitimate journalism.

Our Solution

FactCheck Nepal leverages state-of-the-art GBERT (Nepali BERT) transformer models to automatically detect fake news in Nepali language. Unlike traditional black-box approaches, our system provides explainable AI insights, showing exactly why a news article is classified as real or fake.

Key Features

  • Deep learning-based classification using GBERT architecture
  • Explainable AI with detailed reasoning for each prediction
  • Optimized for Nepali political news content
  • High accuracy with transparent confidence scores
  • Real-time analysis with instant results

Our mission is to empower Nepali citizens with tools to combat misinformation, promote media literacy, and strengthen democratic discourse through transparent, AI-powered fact-checking.

How It Works

Understanding the technology behind fake news detection

1

Input Processing

The system accepts Nepali news articles as text input or URL. The content is preprocessed to clean and normalize the text, handling special characters, punctuation, and formatting specific to Nepali language.

Text Normalization Unicode Handling
2

GBERT Tokenization

The preprocessed text is tokenized using GBERT's vocabulary, which is specifically trained on Nepali corpus. This converts the text into numerical representations that the model can understand.

WordPiece Tokenization Subword Units Position Encoding
3

Feature Extraction

The GBERT transformer model processes the tokenized input through multiple attention layers, extracting deep contextual features and semantic relationships within the text.

Self-Attention Mechanism Contextual Embeddings 12-Layer Transformer
4

Classification

A fine-tuned classification head processes the extracted features to predict whether the news is real or fake, along with a confidence score indicating the model's certainty.

Binary Classification Softmax Activation Confidence Scoring
5

Explainability Analysis

Using attention visualization and feature importance techniques, the system generates human-readable explanations highlighting key phrases, patterns, and linguistic markers that influenced the prediction.

Attention Weights LIME/SHAP Feature Attribution
6

Result Presentation

The final prediction, confidence score, and detailed explanations are presented to the user in an intuitive interface, empowering them to understand the reasoning behind the classification.

Visual Dashboard Interactive UI Export Capability
